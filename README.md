# 📊 پنل کاربری شیک و مدرن ۳X-UI (سنایی)

یک فرانت‌اند مدرن، زیبا و بهینه برای نمایش وضعیت مصرف، زمان باقی‌مانده و حجم کانفیگ‌های کاربران در پنل سنایی (3X-UI).

---

## ✨ ویژگی‌ها
* 📱 طراحی کاملاً واکنش‌گرا (Responsive) و سازگار با موبایل
* ⏱️ نمایش روزهای باقی‌مانده از اعتبار سرویس
* 📊 نمودار مصرف حجم و نمایش پهنای باند مصرفی (آپلود و دانلود)
* 🌐 پشتیبانی از چند سرور (آلمان، ترکیه و...) همراه با نمایش پینگ (Latency)
* 📥 قابلیت دانلود مستقیم برنامه و کپی آسان لینک اتصال / QR Code

---

## 🚀 راهنمای نصب و راه‌اندازی

چون این پروژه یک پنل آماده و استاتیک (تک‌فایل) است، راه‌اندازی آن بسیار ساده است:

### ۱. دریافت فایل پروژه
فایل `index.html` موجود در این ریپازیتوری را دانلود کنید.

### ۲. آپلود روی سرور سنایی
فایل دانلود شده را روی سرور خود آپلود کنید. شما می‌توانید آن را در پوشه دلخواه وب‌سرور خود (مانند Nginx یا Apache) قرار دهید یا مستقیماً در مسیر فایل‌های استاتیک پنل ۳x-ui آپلود کنید.

### ۳. تنظیم مسیر در پنل
پس از آپلود فایل روی سرور، مسیر آن را در بخش تنظیمات پنل ۳x-ui خود ست کنید تا کاربران با وارد کردن آدرس مشخص شده، مستقیماً به این صفحه زیبا هدایت شوند و وضعیت اکانت خود را مشاهده کنند.
