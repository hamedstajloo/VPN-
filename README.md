# 📊 پنل کاربری شیک و مدرن 3X-UI (سنایی)

<p align="center">
  <img src="preview.png" alt="پیش‌نمایش پنل کاربری" width="100%" style="border-radius: 10px; max-width: 600px;">
</p>

یک فرانت‌اند مدرن، زیبا و بهینه برای نمایش وضعیت مصرف، زمان باقی‌مانده و حجم کانفیگ‌های کاربران در پنل سنایی (3X-UI).

---

## ✨ ویژگی‌ها
* 📱 طراحی کاملاً واکنش‌گرا (Responsive) و سازگار با موبایل
* ⏱️ نمایش روزهای باقی‌مانده از اعتبار سرویس
* 📊 نمودار مصرف حجم و نمایش پهنای باند مصرفی (آپلود و دانلود)
* 🌐 پشتیبانی از چند سرور (آلمان، ترکیه و...) همراه با نمایش پینگ (Latency)
* 📥 قابلیت دانلود مستقیم برنامه و کپی آسان لینک اتصال / QR Code

---

## 🚀 راهنمای نصب 

برای دانلود و جایگذاری خودکار این فایل در مسیر مشخص شده، وارد ترمینال سرور خود شوید و دستور زیر را اجرا کنید:

```bash
curl -o /etc/x-ui/index.html [https://raw.githubusercontent.com/hamedstajloo/sanaei-user-panel/main/index.html](https://raw.githubusercontent.com/hamedstajloo/sanaei-user-panel/main/index.html)
 
---

### این نسخه چه فرقی دارد؟
* بخش **«⚙️ راهنمای فعال‌سازی در 3x-ui»** به عنوان یک هدر جداگانه و با فرمت زیبا به انتهای فایل اضافه شده است.
* مراحل سه‌گانه فعال‌سازی (ورود به پنل، بخش Subscription Settings و وارد کردن مسیر در Custom Path) دقیقاً طبق عکس نوشته شده‌اند.
* مسیر نهایی کادر آدرس هم دقیقاً روی همان مقدار مورد نیاز شما یعنی `/etc/x-ui/` تنظیم شده است.
